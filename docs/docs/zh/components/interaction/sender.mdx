# Sender 输入框

输入框是一个 textarea 组件，用于输入和发送消息。

## 示例

```tsx preview
import { Sender } from "@matechat/react";
import { useCallback, useState } from "react";

export default function () {
  return (
    <div className="w-full p-1">
      <Sender placeholder="请输入消息" />
    </div>
  );
}
```
