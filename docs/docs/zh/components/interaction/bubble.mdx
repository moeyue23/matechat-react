# Bubble 气泡

Bubble 组件用于承载对话内容的气泡组件，支持透明背景、自定义加载动画等。

## 代码演示

```tsx preview
import { Bubble } from "@matechat/react";

export default function () {
  return (
    <div className="flex flex-col w-full">
      <Bubble text="你好，有什么我可以帮助你的吗" />
    </div>
  );
}
```
